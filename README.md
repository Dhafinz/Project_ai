# Model Prediksi Harga Rumah
Model Prediksi Harga Rumah adalah sebuah proyek yang bertujuan untuk memprediksi harga rumah berdasarkan berbagai fitur seperti lokasi, ukuran, jumlah kamar, dan lain-lain. Proyek ini menggunakan teknik machine learning untuk menganalisis data dan menghasilkan prediksi yang akurat.

## Fitur Utama
- Prediksi harga rumah berdasarkan berbagai fitur.

## Cara Menjalankan Proyek
1. Clone repository ini ke komputer Anda.
2. Buat lingkungan virtual (virtual environment) untuk proyek ini menggunakan perintah `python -m venv env`.
3. Aktifkan lingkungan virtual yang telah dibuat:
   - Untuk Windows: `.\env\Scripts\activate`
   - Untuk macOS/Linux: `source env/bin/activate`
4. Install semua dependensi yang diperlukan menggunakan perintah `pip install -r requirements.txt`.
5. Jalankan file utama untuk memulai prediksi harga rumah.
    - Gunakan perintah `python main.py` untuk menjalankan proyek.
6. Jalankan streamlit untuk melihat hasil prediksi secara interaktif.
    - Gunakan perintah `streamlit run app.py` untuk menjalankan aplikasi Streamlit.
